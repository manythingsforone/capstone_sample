신호등과 같이 업비트와 바이낸스 거래소의 비트코인 가격 차이를 색으로 시각화 함.

한국 최대 규모의 거래소의 업비트의 가이드라인을 따라 차이가 3%를 넘을 경우, 주황색으로 5%를 넘을 경우 빨간색으로 시각화 함으로

가상화폐 투자에 도움이 될 수 있도록 직접적인 시각화를 함.
